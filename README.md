# 🏡 Kas Estate - Modern Real Estate Platform

A full-stack real estate application built with **Next.js 14**, **PostgreSQL**, and **Tailwind CSS**.

### 🚀 Features
- **Admin Dashboard:** Full CRUD (Create, Read, Update, Delete) for property listings.
- **Image Uploads:** Powered by UploadThing.
- **Authentication:** Secure Admin login via Clerk.
- **Database:** Hosted on Neon (Serverless Postgres).

### 🛠️ Tech Stack
- **Framework:** Next.js (App Router)
- **Database:** Neon DB
- **Auth:** Clerk
- **Storage:** UploadThing
- **Icons:** Lucide React
